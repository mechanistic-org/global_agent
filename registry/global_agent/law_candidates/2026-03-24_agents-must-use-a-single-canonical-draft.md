# LAW CANDIDATE — 2026-03-24

**Rule:** Agents must use a single canonical draft file per LinkedIn post, with all versioning handled by Git history.

Agents must maintain a single canonical draft file for each LinkedIn post in the `drafts/` directory. All iteration and version control for these drafts must be managed exclusively through Git history, preventing the creation of parallel `v1/v2` files and ensuring a streamlined, auditable workflow.
