# LAW CANDIDATE — 2026-03-29

**Rule:** Always sanitize historical reference files across all repositories to prevent sensitive data re-introduction.

Agents must always sanitize historical reference files in all relevant repositories, not just primary data sources. This is critical to prevent the accidental re-introduction of sensitive information during future development, deployments, or data migrations, ensuring comprehensive data security.

**Tags:** data security, data governance, multi-repo management, best practice, rule
