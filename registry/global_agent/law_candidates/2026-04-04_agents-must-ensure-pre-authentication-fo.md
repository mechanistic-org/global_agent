# LAW CANDIDATE — 2026-04-04

**Rule:** Agents must ensure pre-authentication for external service dashboards when performing automated UI tasks.

Agents must ensure pre-authentication for external service dashboards when performing automated UI tasks. If a browser subagent is dispatched to navigate a dashboard like Cloudflare Zero Trust, it must not be blocked by a login screen, requiring prior authentication in its default browser data.
