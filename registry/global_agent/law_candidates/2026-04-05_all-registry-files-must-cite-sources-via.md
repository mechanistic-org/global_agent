# LAW CANDIDATE — 2026-04-05

**Rule:** All registry files must cite sources via a `sources: []` array in frontmatter, enforcing provenance.

To eliminate un-cited claims, a rule is established to enforce strict provenance metrics: all generated registry files must include a `sources: []` array in their frontmatter, citing absolute URIs or commit hashes of informing core codebase files, inspired by DocMason's evidence-first approach.
