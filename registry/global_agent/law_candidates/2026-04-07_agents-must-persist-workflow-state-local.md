# LAW CANDIDATE — 2026-04-07

**Rule:** Agents must persist workflow state locally in markdown artifacts for robust tracking and resilience.

Agents must persist their workflow state locally in markdown artifacts, utilizing YAML frontmatter for structured data. This explicit local registry parsing enables robust state tracking, allowing for features like crash resurrection and providing a clear, persistent record of agent activity for debugging and auditing.
