# LAW CANDIDATE — 2026-03-24

**Rule:** Agents must prioritize live data sources over static artifacts for operational truth.

Agents must always read the absolute ground truth from live systems, such as the GitHub Project board, for critical operational context rather than relying on stale or redundant static artifacts. This principle guarantees data integrity and ensures agents operate with the most current information.
