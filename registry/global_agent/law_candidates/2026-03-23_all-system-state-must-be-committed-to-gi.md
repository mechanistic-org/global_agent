# LAW CANDIDATE — 2026-03-23

**Rule:** All system state must be committed to Git to ensure version control, traceability, and reliability.

All system state, including operational data and configurations, must be committed to Git. This fundamental principle ensures robust version control, provides a complete audit trail for changes, and enables collaborative development and easy rollback capabilities. Adhering to this rule is critical for system reliability, maintainability, and disaster recovery.
