  and maintainability.

The `$ENOS_ROOT` environment variable has been formally defined and integrated across the codebase, replacing all previously hardcoded paths. This architectural decision enhances system portability, simplifies environment configuration, and significantly improves overall code maintainability for the EN-OS project.
