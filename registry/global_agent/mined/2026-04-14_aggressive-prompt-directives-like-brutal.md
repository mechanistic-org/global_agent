---
title: Aggressive prompt directives like "brutalist bullets" led to pretentious, unprofessional formatting in agent outputs.
date: 2026-04-14
context_node: conversation_miner
---

A critical lesson learned is that overly aggressive prompt directives, such as instructing the LLM to write "brutalist, 'Hard Ore' bullets," can lead to unintended and unprofessional output formatting. This specific instruction resulted in pretentious incident-report structures like **The Trigger:**, **The Intervention:**, and **The Result:**, highlighting the need for precise and nuanced prompt engineering to achieve desired stylistic outcomes.
