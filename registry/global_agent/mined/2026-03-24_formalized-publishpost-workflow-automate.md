A new, formalized workflow (`.agent/workflows/publish_post.md`) defines the ritual for publishing LinkedIn posts. This workflow automates steps such as stripping frontmatter, prompting for the live URL, updating file status, moving files, displaying self-comments, updating thread ledgers, and committing changes, significantly reducing manual errors and ensuring consistency.
