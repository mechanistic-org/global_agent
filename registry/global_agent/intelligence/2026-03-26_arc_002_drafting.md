# Arc 002 Drafting — The Nervous System

## Overview
This document formalizes the strategy, sequencing, and drafts for **Arc 002: The Nervous System**. If Arc 001 built the skeleton and taught the machine to remember (Memory, Substrate, Constraints), Arc 002 teaches the machine to wake itself up. It transitions the architecture from a human-triggered pull system into an event-driven operating system.

## Arc Sequence

### Post 1: The Starter Motor
**Core Move:** Names the gap Arc 001 left — the machine works, but only when a human starts it.
**Summary:** Maps out the 7 layers of the EN-OS (Substrate, Ingress, Router, Sandbox, Brain, Gatekeeper, Cache) and introduces the problem: a factory that waits for you to turn the crank isn't an OS, it's a very precise tool.
**Comment Strategy:** Retroactive Arc 001 stitch + polling trap tease.

### Post 2: Scheduled Amnesia
**Core Move:** Destroys the naive fix (cron jobs).
**Summary:** Explains why polling a sprint board is architecturally weak and introduces the CNC limit switch manufacturing metaphor. Explains why we need an event-driven nervous system to cross the OS threshold.
**Comment Strategy:** Cron job failure story + event vs. time pivot.

### Post 3: Wiring the Limit Switch
**Core Move:** The technical build post.
**Summary:** Details the actual architecture that makes autonomy possible: GitHub Webhooks -> Cloudflare Tunnels -> NanoClaw FastAPI Router. Explains how the safety constraint cages built in Arc 001 are the only reason this dangerous connectivity is allowed, ensuring the ephemeral sandbox burns down clean.
**Comment Strategy:** The Ouroboros webhook-loop war story + dropping the `nanoclaw-router` filepath as proof.

### Post 4: The Lights-Out Run
**Core Move:** Proof the nervous system works using the live portfolio.
**Summary:** The first time the system completed a full documentation cycle entirely offline (updating the Colophon site autonomously). Physical proof that the machine senses reality and ships the exhaust without asking for human permission.
**Comment Strategy:** The silence of execution + live link to the Colophon update.

### Post 5: The Single-Threaded Factory
**Core Move:** The honest gap + Arc 003 handoff.
**Summary:** While the system works autonomously, it is entirely single-threaded and struggles with complex multi-disciplinary orchestration (e.g. BOM cascade mapping). Transition into exactly why a real, chaotic product build is needed to stress-test swarm orchestration. 
**Comment Strategy:** Two-arc comprehensive stitch + teasing MootMoat (the Arc 003 case study).
