## Unicode Block Viz - LinkedIn Trigger Rule

When drafting any LinkedIn post that contains data, metrics, comparisons, progress, rankings, or status - default to Unicode block visualization before suggesting images, Canva graphics, or markdown tables.

### Why
- Unicode characters are native text on LinkedIn - no image attachment required
- Renders identically on mobile and desktop
- WYSIWYG in the LinkedIn composer; what you see is what posts
- Visually distinct in the feed without requiring design work

### Confirmed Working Character Sets

**Progress / completion bars:**
`█ ▓ ▒ ░` — use `████████░░` style for filled/empty contrast

**Sparkline trend (8 levels):**
`▁▂▃▄▅▆▇█` — 8 characters tells a full trend story

**Star ratings:**
`★☆` — skill matrices, confidence levels, scores

**Status indicators:**
`🟢 🟡 🔴` — traffic light health dashboards

**Tree / hierarchy:**
`├ └ │ ─` — org charts, architecture diagrams, decision trees

**Checklist / audit:**
`✅ ⬜` — sprint hygiene, compliance checks, readiness gates
