# The Moves
A collection of "signature interactions" that elevate the portfolio from a static archive to a living machine.

## Active Moves

### 1. The D3licious Dataviz (DataSheet)
*   **Description:** Animated D3.js charts that "grow" on verified data.
*   **Status:** Active (SkillGraph, MetricComparison).
*   **Vibe:** Quantitative, precise, "The Engineer".

### 2. The Hero Wiggle (3D)
*   **Description:** Model Viewer camera orbit reacts to mouse position with subtle parallax + idle breathing.
*   **Status:** Active (Hyperspace).
*   **Vibe:** Alive, tangible, "The Machine".

### 3. Elastic Inspection (3D)
*   **Description:** When the user rotates a model, it "snaps back" to the curated angle after a delay.
*   **Status:** Active.
*   **Vibe:** Curated, guided.

### 4. Cinematic Breathe (Hero)
*   **Description:** The background hero image slowly zooms in/out (Scale 1.05 <-> 1.15) over 20s.
*   **Status:** Active (Hyperspace).
*   **Vibe:** Atmospheric, epic.

### 5. Parallax Depth (Hero)
*   **Description:** The hero image moves slower than the scroll (TranslateY), creating deep Z-space.
*   **Status:** Active.
*   **Vibe:** Immersion.

## The Vault (Planned/Dormant)

### 6. Text Glitch (The Ghost)
*   **Description:** Random character replacement on hover or load (Matrix style).
*   **Status:** Dormant.
*   **Vibe:** Cyberpunk, raw, "The Code".

### 7. Swarm Physics (The Hive)
*   **Description:** Particles that flee the mouse cursor.
*   **Status:** Prototype (Hyperspace Background).
